[[Update Upgrade]]

Install snap package manager:
https://snapcraft.io/docs/installing-snap-on-arch-linux

Packages:
- telegram-desktop
- chromium (or *pamac install google-chrome*)

List of installed:
```shell
snap list
```

### Update All Snaps:
```shell
sudo snap refresh
```
